### Webhook Signature Verification

All webhook events include a signature for security. To verify authenticity:

```python
import hmac
import hashlib

WEBHOOK_SECRET = '<YOUR_WEBHOOK_SECRET>'
raw_payload = request.get_data(as_text=True)
received_signature = request.headers.get("X-Signature", "")

expected_signature = hmac.new(
    WEBHOOK_SECRET.encode(),
    raw_payload.encode(),
    hashlib.sha256
).hexdigest()

if hmac.compare_digest(received_signature, expected_signature):
    print("Webhook verified successfully!", data)
else:
    print("Invalid signature")
```

**Note:** All `event_date` values are in UTC.

## Event Types

| Event Type             | Description                                      |
| :--------------------- | :----------------------------------------------- |
| `ProspectUpdate`       | Prospect created or updated with new information |
| `Handoff`              | Leasing agent intervention required              |
| `ProspectEmailEvent`   | Email sent between Elise and prospect            |
| `ProspectTextEvent`    | Text message sent between Elise and prospect     |
| `ProspectVoiceAIEvent` | Prospect called VoiceAI                          |
| `ProspectOptInEvent`   | Prospect opted into communication channel(s)     |
| `ProspectOptOutEvent`  | Prospect opted out of communication channel(s)   |
